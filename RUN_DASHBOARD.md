## Prerequisites

Make sure you have installed all dependencies:

```bash

```

Required packages include:
- Flask 3.0.0
- flask-cors 4.0.0
- requests
- PyGithub
- All other packages from requirements.txt

## Using the Dashboard

1. **Open your browser** and navigate to `http://localhost:5000`
2. **Enter a task** in the text area (e.g., "Create a user authentication module")
3. **Select language** (Python, JavaScript, TypeScript, etc.)
4. **Select project type** (Module, API, Web App, Library, etc.)
5. **Click "âœ¨ Generate Code"** to start the Round Table discussion
6. **View the results**:
   - Individual AI model recommendations
   - Consensus code generated by all 4 AI models
   - Discussion summary
   - Final recommendations

## API Endpoints

The dashboard provides several API endpoints:

### `GET /api/health`
Health check endpoint
```bash
curl http://localhost:5000/api/health
```

### `GET /api/examples`
Get example tasks
```bash
curl http://localhost:5000/api/examples
```

### `GET /api/stats`
Get usage statistics
```bash
curl http://localhost:5000/api/stats
```

### `GET /api/history`
Get generation history
```bash
curl http://localhost:5000/api/history
```

### `POST /api/discuss`
Start a Round Table discussion
```bash
curl -X POST http://localhost:5000/api/discuss \
  -H "Content-Type: application/json" \
  -d '{"task": "Create a hello world function", "language": "python", "project_type": "module"}'
```

## Troubleshooting

### Port 5000 Already in Use

If you get an error about port 5000 being in use:

```bash
# Find the process using port 5000
lsof -ti:5000

# Kill the process (replace PID with actual process ID)
kill -9 <PID>
```

### Module Not Found Error

Make sure all dependencies are installed:

```bash
pip install -r requirements.txt
```

### Templates Not Found

The dashboard will automatically create the templates directory if it doesn't exist. The `dashboard.html` template should be in `templates/dashboard.html`.

## Running in Production

For production deployments, use a production WSGI server:

```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 dashboard:app
```

And set a secure secret key:

```bash
export FLASK_SECRET_KEY="your-secure-random-string-here"
```
